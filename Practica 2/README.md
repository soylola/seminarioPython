# Explicación de cómo instalar las dependencias y ejecutar el programa.

Para este proyecto se necesitarán dos dependencias: Python 3.12 y Jupyter Notebook 4.5.6. A continuación se detalla una guía de instalación y ejecución:

1. PYTHON
En el siguiente link a la página oficial de Python podrán encontrar la versión 3.12 para cada sistema operativo: python.org/downloads
Una vez descargado el archivo .exe dar a "next" hasta finalizar instalación.

2. JUPYTER NOTEBOOK
Para instalar Jupyter Notebook debemos abrir la terminal y ejecutar el comando "pip install notebook". Comenzarán a descargarse los archivos necesarios y una vez finalizada la instalación, con el comando "jupyter notebook" se abrirá una pestaña de jupyter en el navegador. Debemos tener en cuenta que no podemos cerrar la terminal mientras se ejecute jupyter notebook. Una vez abierta la pestaña en el navegador elegimos el archivo con extensión .ipynb que querramos ejecutar.
